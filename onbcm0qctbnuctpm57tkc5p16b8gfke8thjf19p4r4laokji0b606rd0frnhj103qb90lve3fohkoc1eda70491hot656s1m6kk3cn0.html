doc

Tuples are represented as right associative heterogeneously typed lists.

There is a useful syntactic sugar for tuples:

- `(a, b)` on the term level is syntactic sugar for `base.Tuple a (base.Tuple b ())`

- `(a, b)` on the type level is syntactic sugar for `base.Tuple a (base.Tuple b ())`

You can pattern match on tuples using this syntax, or using the base.Tuple constructor itself:
