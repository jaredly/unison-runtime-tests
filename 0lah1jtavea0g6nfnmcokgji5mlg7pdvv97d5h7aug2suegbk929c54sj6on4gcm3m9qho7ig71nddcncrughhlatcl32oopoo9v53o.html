doc

base.Bag t is the type of multisets of elements of type t.

A base.Bag is like a base.List where elements occur in no particular order, or like a base.Set where elements can occur more than once.

Constructing bags

base.Bag a

The empty base.Bag.

k -> base.Bag k

base.Bag.singleton k constructs a base.Bag with a single element k occurring exactly once.

[k] -> base.Bag k

base.Bag.fromList ks creates a new base.Bag from the base.List ks. The resulting base.Bag contains exactly as many occurrences of each element as there are in the base.List ks.

[(a, base.Nat)] -> base.Bag a

Constructs a base.Bag from a base.List of elements paired with the number of times they should occur in the base.Bag.

base.Text -> base.Bag base.Char

base.Bag.fromText t is the base.Bag of characters in the base.Text t.

Adding and removing elements

k -> base.Bag k -> base.Bag k

base.Bag.add k b adds one occurrence of the element k to the base.Bag b. The rule is that if k occurs n times in the base.Bag b then it will occur n + 1 times in base.Bag.add k b.

base.Nat -> k -> base.Bag k -> base.Bag k

base.Bag.addN n k b adds n occurrences of the element k to the base.Bag b.

k -> base.Bag k -> base.Bag k

base.Bag.remove k b removes one occurrence of the element k from the base.Bag b if k occurs in the base.Bag b. Otherwise it does nothing.

base.Nat -> k -> base.Bag k -> base.Bag k

base.Bag.removeN n k b removes n occurrences of the element k from the base.Bag b if k occurs in it. If the base.Bag has fewer occurrences of k, then base.Bag.removeN removes them all. If k doesn't occur in the base.Bag, it does nothing.

k -> base.Bag k -> base.Bag k

base.Bag.removeAll k b removes all occurrences of the element k from the base.Bag b if k occurs in the base.Bag b. Otherwise it does nothing.

(a ->{e} base.Boolean) -> base.Bag a ->{e} base.Bag a

base.Bag.removeWhere p b removes the elements from the base.Bag b for which the function p returns true.

(i ->{e} base.Boolean) -> base.Bag i ->{e} base.Bag i

base.Bag.filter p b returns a new base.Bag with just the elements from b for which the function p returns true.

Querying and counting elements

base.Bag a -> a -> base.Boolean

base.Bag.contains b k is true if the value k is an element of the bag b, and false otherwise.

a -> base.Bag a -> base.Nat

base.Bag.count e b returns the number of times the element e occurs in the base.Bag b.

a -> base.Bag a -> base.Boolean

base.Bag.elementOf k b is true if the value k is an element of the bag b, and false otherwise.

base.Bag a -> base.Boolean

base.Bag.isEmpty b is true if the base.Bag b has no elements, and false otherwise.

base.Bag a -> base.Nat

base.Bag.size b is the number of elements in the base.Bag b, including duplicate elements.

For example, base.Bag.size (base.Bag.fromList [2,3,1,1]) == 4.

Combining bags

base.Bag a -> base.Bag a -> base.Bag a

addAll b1 b2 (or equivalently b1 + b2) constructs a new base.Bag with elements from b1 and b2. The number of times each element occurs in the new base.Bag is the sum of the number of times it occurs in b1 and the number of times it occurs in b2.

For example, if

a = Bag.fromList [2,3,1,1]
b = Bag.fromList [1,3,3,1,1]

then

addAll a b = Bag.fromList [3,1,1]

(i ->{g} a ->{g} t) -> base.Bag i -> base.Bag a ->{g} base.Bag t

base.Bag.convolve f bx by applies the function f to every pair of elements (x,y) where x comes from the base.Bag bx and y comes from the base.Bag by. This is called the convolution of b1 and b2 with the function f.

For example, if

bx = base.Bag.fromText "foo"
by = base.Bag.fromText "off"

then

base.Bag.occurrenceList (base.Bag.convolve (x y -> base.Text.fromCharList [x, y]) bx by) =
   [("ff", 2), ("fo", 1), ("of", 4), ("oo", 2)]

base.Bag a -> base.Bag a -> base.Bag a

base.Bag.difference b1 b2 removes elements from the base.Bag b1 to the extent that they exist in the base.Bag b2. If an element occurs n times in b1 and m times in b2, it will occur base.Nat.drop n m times in the result.

For example, if

b1 = base.Bag.fromList [2,3,1,1]
b2 = base.Bag.fromList [1,3,3,4]

then

base.Bag.difference b1 b2 = base.Bag.fromList [2,1]

base.Bag a -> base.Bag a -> base.Bag a

base.Bag.intersect b1 b2 constructs the intersection of b1 and b2, which is the largest base.Bag that is a base.Bag.subbag of both b1 and b2. This is a new base.Bag with elements from b1 and b2 where the number of times each element occurs in the new base.Bag is either the number of times it occurs in b1 or the number of times it occurs in b2, whichever is less.

For example, if

a = Bag.fromList [2,3,1,1]
b = Bag.fromList [1,3,3,1,1]

then

intersect a b = Bag.fromList [3,1,1]

base.Bag a -> base.Bag b -> base.Bag (a, b)

The expression product bx by (or equivalently bx * by) is the base.Bag of all pairs of elements (x,y) where x comes from the base.Bag bx and y comes from the base.Bag by. The number of times (x,y) occurs in the result is x * y.

base.Bag a -> base.Bag a -> base.Bag a

base.Bag.union b1 b2 constructs the union of b1 and b2, which is the smallest base.Bag that is a base.Bag.superbag of both b1 and b2. This is a new base.Bag with elements from b1 and b2 where the number of times each element occurs in the new base.Bag is either the number of times it occurs in b1 or the number of times it occurs in b2, whichever is greater.

For example, if

a = Bag.fromList [2,3,1,1]
b = Bag.fromList [1,3,3,1,1]

then

union a b = Bag.fromList [2,3,3,1,1,1]

Comparing bags

base.Bag a -> base.Bag a -> base.Boolean

base.Bag.from b1 b2 returns true if every element in b1 occurs at least once in b2.

base.Bag a -> base.Bag a -> base.Boolean

base.Bag.subbag b1 b2 returns true if no element occurs more times in the base.Bag b1 than in the base.Bag b2.

base.Bag a -> base.Bag a -> base.Boolean

base.Bag.superbag b1 b2 returns true if no element occurs more times in the base.Bag b2 than in the base.Bag b1.

Conversions to other structures

base.Bag a -> base.Map a base.Nat

base.Bag.counts b returns a base.Map where the keys are the elements of the base.Bag b and the value under each key is the number of times that element occurs in b.

For example, if

b = base.Bag.fromText "abracadabra"

then

base.Bag.counts b = base.Map.fromList [(?a, 5), (?b, 2), (?c, 1), (?d, 1), (?r,

2)]

base.Bag a -> [(a, base.Nat)]

base.Bag.occurrenceList b returns a list of all the elements of the base.Bag b, paired with the number of times that element occurs in b.

base.Bag a -> [a]

base.Bag.toList b returns the list of all the elements of the base.Bag b, including duplicates.

base.Bag base.Char -> base.Text

base.Bag.toText b takes a base.Bag of characters and returns base.Text with those characters in an unspecified order.

base.Bag a -> base.Set a

base.Bag.toSet b returns the base.Set of elements in the base.Bag b.

Traversal and iteration

(a ->{e} base.Bag b) -> base.Bag a ->{e} base.Bag b

base.Bag.flatMap f b applies the function f to every element of the base.Bag b. The function f returns a whole base.Bag each time, and base.Bag.flatMap collects all their elements into one base.Bag.

For example, if

bag = base.Bag.fromList ["site", "real", "live"]

then

base.Bag.toText (base.Bag.flatMap (n -> base.Bag.fromText n) bag) =
  "aeeeiillrstv"

(b ->{g} a ->{g} b) -> b -> base.Bag a ->{g} b

base.Bag.foldLeft f z b iterates over the elements of the base.Bag b, applying the binary operator f to each element and the result so far. The result starts as the value z, which is also returned in case the base.Bag is empty.

(a ->{g} b ->{g} b) -> b -> base.Bag a ->{g} b

base.Bag.foldRight f z b iterates over the elements of the base.Bag b, applying the binary operator f to each element and the result so far. The result starts as the value z, which is also returned in case the base.Bag is empty.

((a, base.Nat) ->{e} (b, base.Nat)) -> base.Bag a ->{e} base.Bag b

base.Bag.modify f b passes each unique element in the base.Bag b, together with its count in b, to the function f which returns a modified element and a new count. Can be used to modify elements, their counts, or both.

(a -> b) -> base.Bag a -> base.Bag b

base.Bag.map f b applies the function f to every element of the base.Bag b, constructing a new base.Bag of the results. The function is not allowed to use abilities. For a version of this that allows abilities, see base.Bag.traverse.

For example, if

b = base.Bag.fromList [2,3,1,1]

then

base.Bag.map base.Nat.isEven b = base.Bag.fromList [true, false, false, false]

(a ->{e} b) -> base.Bag a ->{e} base.Bag b

base.Bag.traverse f b iterates over the elements of the base.Bag b, applying the function f to each element, collecting the results in a new base.Bag.

Operations on element counts

((a, base.Nat) ->{e} (b, base.Nat)) -> base.Bag a ->{e} base.Bag b

base.Bag.modify f b passes each unique element in the base.Bag b, together with its count in b, to the function f which returns a modified element and a new count. Can be used to modify elements, their counts, or both.

base.Nat -> base.Bag a -> base.Bag a

base.Bag.scale k b constructs a new base.Bag with the same elements as b, but where the number of times each element occurs is multiplied by k.

For example, if

 b = Bag.fromList [2,3,1,1]

then

 scale 2 b = Bag.fromList [2,2,3,3,1,1,1,1]

(base.Nat ->{e} base.Nat) -> base.Bag a ->{e} base.Bag a

base.Bag.mapCounts f b constructs a new base.Bag with the same elements as b, but with the number of times any given element occurs modified by the function f.
